# 步骤

## 项目初始化

````md
npm init -y

npm add -D vitepress

npx vitepress init
````

## 代码提交

增加.npmrc 文件
创建 .gitignore
提交代码


## VitePress 配置

修改 .vitepress/config.mjs，参考：[主题配置](https://vitepress.dev/zh/reference/default-theme-config)

## 配置 icon、logo、首页右边大图

把 src/public/ 目录修改为与 src/ 目录同级别：[publicdir](https://cn.vitejs.dev/config/shared-options#publicdir)
